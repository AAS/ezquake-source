EzQuake EzHash - version 1.2
Created by AAS, 2009-2011


You must install python for this script to work.
Allowed extensions are: "bsp", "lmp", "md3", "mdl", "spr", "wav"


How to obtain the crc of a file:
1. Make sure Python is installed
2. Copy ezhash.py and your file of choice into a directory, for example C:\hashme
3. Open command prompt and navigate to C:\hashme
4. Type: ezhash.py c:\hashme NAME_OF_FILE    !! DO NOT INCLUDE FILE EXTENSION !!
