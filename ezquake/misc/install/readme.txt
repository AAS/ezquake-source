Select your Quake directory as the target directory for the installation.

You need original Quake 1 to use the ezQuake client.
Consult http://ezQuake.SourceForge.net/docs/?install for further help.

Note: This installer will NOT alter your other QuakeWorld clients settings.
It will NOT install any additional graphics, unless you explicitly allow it.
This installer will install the smallest amount of data
required to launch ezQuake on your computer.
